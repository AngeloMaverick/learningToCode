==============[NPM COMMANDS]==============
npm start               --> Inicializa a aplicação
npm init -y             --> Criação do arquivo package.json
npm install             --> Instalação do node_modules
npm i json-server       --> Instalação do json-server (node_modules + package-lock.json)


==============[NPM ANGULAR COMMANDS]==============
npm i -g @angular/cli        --> Instalação do Command Line Interface (CLI) do Angular
ng                           --> help Angular
ng new "minha-app"           --> Criar um projeto Angular
ng new "minha-app" --minimal --> Cria um projeto sem arquivos desnecessários para um projeto pequeno
ng add @angular/material     --> Instalar os componentes do material
ng generate component "localização do path" (exemplo components/template/header) --> Criação de componentes header, footer e nav
